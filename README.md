# excel-academic-dashboard
Un petit dashboard personnel pour mes examens

## Description

This project is an academic dashboard built in Microsoft Excel to help students track their academic performance and preparation for exams.

The goal of the dashboard is to visualize key indicators such as progress level, remaining study time, and exam preparation status.

## Features

- Academic progress tracking
- Visual indicators for preparation level
- Automatic calculations using Excel formulas
- Dashboard layout for quick overview

## Technologies Used

- Microsoft Excel
- Data visualization
- Formula-based calculations

## My Role

I designed the dashboard structure, created the formulas used for automatic calculations, and organized the visual layout to make the information easy to understand.

## Learning Outcomes

Through this project I improved my understanding of data organization, logical formulas, and visual presentation of information.
